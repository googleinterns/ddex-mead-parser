# C++ DDEX XML to Protocol Buffer
**[ERN + MEAD] Message to Protocol Buffer Conversion Package**

### Build
Run `bazel build //ddex-converterli:converter_cli`

### Dependencies

- pugixml
- cxxopts
- google protobuf
